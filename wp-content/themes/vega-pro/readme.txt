== Description ==

Vega Pro is a clean, blog-focused theme. It also has a custom frontpage template to showcase special timeless elements/information. It also doubles as a one-page theme if needed. Vega Pro is versatile since it can be used for large and small websites alike, such as landing pages, blogs, company websites, and personal web pages. Vega Pro features:

* Ability to create a one-page theme
* Three pre-built color schemes that can be switched/chosen from the Customizer
* Unlimited color choices
* Font choices (Google Fonts)
* Four blog feed views (Excerpt Only, Small Image on Left Excerpt on Right, Large Image and Full Content, and no sidebar with posts shown in two columns)
* Ability to turn sidebar on or off on pages and posts
* Custom front page template that allows for multiple sections such as a call to action row, featured pages with images, featured pages with icons, recent blog posts, testimonials, team, logos/partners, and open content

For a complete list of features and detailed documentation, please go to http://www.lyrathemes.com/documentation/vega-pro.pdf
